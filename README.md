# Javeria Fatima Asif — Portfolio

A Minecraft-inspired, fully animated personal portfolio built with vanilla HTML, CSS, and JavaScript — no frameworks, no build step.

🔗 **Live site:** (https://javeriafatimaasif.github.io/portfolio/)

## About

Software Engineering student at Mehran University of Engineering and Technology (MUET), Jamshoro, with a 4.0/4.0 GPA. Full-stack developer working primarily in Python/Flask, and a published author writing since age 9.

This site showcases my projects, skills, education, and publications through a blocky, voxel-style theme with:

- 🌞 Day/night toggle
- 🧊 Falling-block name animation
- 🎒 Inventory-style skills section
- 📦 Clickable project "chests"
- 🗺️ Timeline-style education & experience path
- 🏆 Scroll-triggered achievement toasts
- 📚 Publications bookshelf

## Tech Stack

- HTML5
- CSS3 (custom animations, no frameworks)
- Vanilla JavaScript (IntersectionObserver for scroll effects)
- Google Fonts (Press Start 2P, VT323, Inter)

## Running Locally

Clone the repo and open `index.html` directly in a browser — no build tools or dependencies required.

```bash
git clone https://github.com/javeriafatimaasif/portfolio.git
cd portfolio
open index.html
```
